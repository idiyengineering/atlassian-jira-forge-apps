# Atlassian Jira Fields Viewer

[![License](https://img.shields.io/badge/license-Apache--2.0-green.svg)](LICENSE)

**Atlassian Jira Fields Viewer** is a free Atlassian Forge app that allows you to view and explore all custom and system fields available in your Jira instance.  
It's particularly useful for Jira admins, developers, and automation engineers who need quick visibility into field configurations.

---

## üöÄ Features

- üìã Lists all Jira fields (including system and custom fields)
- üîç Displays field name, field ID, field type, and associated project (if applicable)
- üß† Automatically detects team-managed (next-gen) field scopes and maps them to project names
- üìä Clean and sortable table UI built with Forge + React
- üîí Runs securely within Atlassian Forge infrastructure
